About

HTML, or Hypertext Markup Language, is a fundamental language for building and structuring web pages. However, due to some complexities revolving around programming or makrup languages, it could be difficult to get started. Especially for me, whose learning ability stems from self-learning and testing limits, many HTML guides which limited my creativity in the learning phase obstructed my abilities. My goal for this website is to teach the basics of using HTML and CSS, and letting students experiment with the functions I give them, unlike many other strict tutorials online.
