About

I recently built a website to teach others HTML as part of my personal project for MYP. HTML, or Hypertext Markup Language, is a fundamental language for building and structuring web pages. My website includes a series of lessons that cover the basics of HTML, from setting up a web page to adding text, images, and links. I also include examples and exercises for users to practice what they've learned. I believe that my website is a great resource for anyone looking to learn HTML and start building their own web pages.
