Intro to HTML

HTML (HyperText Markup Language) is a standard markup language used for creating and structuring web pages. It is the foundation of most websites, and is used to define the content and structure of web pages, as well as the relationships between different elements on a page.



Intro to CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the look and formatting of a document written in a markup language. It is most commonly used with HTML to style web pages, but can also be used with other markup languages such as XML.
